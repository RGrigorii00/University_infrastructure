### WB_parser
Тема работы:
* РАЗРАБОТКА ИНФОРМАЦИОННОЙ СИСТЕМЫ УПРАВЛЕНИЯ И АДМИНИСТРИРОВАНИЯ КОМПЬЮТЕРНЫМИ РЕСУРСАМИ В ФГБОУ ВО ПЕРМСКИЙ ГАТУ, Г. ПЕРМЬ

Система кросплатформенна (Windows, Linux), внутри системы есть множество функции: сбор информации о рабочей станции; библиотека приложений с возможностью установки, удаления, обновления; система заявок (тикетов); понятное отображения всей статистики в дашбордах Grafana; локализация на два языка (Английский и Русский) и общие настройки приложения.

В дальнейшем планировалось реализовать удаленный доступ к станции через Rust desk.
